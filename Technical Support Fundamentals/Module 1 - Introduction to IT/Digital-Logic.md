## What learned

### Computer language
Computers process and tranfer data in a language known as binary. It's essentialy just a set of 0s and 1s. Data is typically stored in a byte, which is 8 bits, or 8 0s or 1s. This allows for 256 possible combinations of a byte. Computer can then encode or decode these 
bytes into color, letters, numbers and even emojis. In the present-day, they use something known as UTF-8 which essentially a dictionary for the computer to use. It is built off an older "dictionary" called ASCII which translated the English alphabet and punctuation. 
Computers advance to using UTF-8 because it could store data in multiple bytes. This allowed for values that went beyond 256 like emojis.
